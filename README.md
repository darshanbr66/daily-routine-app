# Daily Routine & Productivity Application

A production-ready Daily Routine & Productivity platform built using the MERN Stack.

## Tech Stack

### Frontend

- React (Vite)
- Tailwind CSS
- React Router
- Axios

### Backend

- Node.js
- Express.js

### Database

- MongoDB Atlas
- Mongoose

### Authentication

- JWT Authentication
- Protected Routes
- Password Hashing (bcrypt)
